# Next_gorugo

# Overview 
Next.jsで開発したポートフォリオ。

Next.js難しい。そしてまだコードが拙い。

## Requirement
### OS 
- MacOS Ventura 13.0 (確認済)
## Library
- Next.js
- Tailwind CSS

## Installation＆Usage
1. Clone this repository
```
git clone git@github.com:GoRuGoo/Next_gorugo.git
```
2. Install Library
```
npm install
```
3. Start Locally
```
npm run dev
```
